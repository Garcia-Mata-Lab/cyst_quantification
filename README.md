# Code and quantitative pipelines for [Cyst quantification]

This repository contains Fiji/ImageJ macros and R scripts used for image analysis and quantitative data analysis in:

**[Lovejoy, M. et al., 2026]**  
*[SGEF, a RhoG-specific GEF, influences lumen morphology and collective cell migration in 3D epithelial cysts]*

- `fiji/` – Fiji/ImageJ macros for 
- `r/` – R scripts for statistical analysis and plotting
